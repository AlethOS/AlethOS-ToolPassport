# Dashboard Agent Instructions

- Use Next.js, React, Tailwind, React Flow, and TanStack Query.
- The dashboard presents backend data and sends user actions only.
- Do not duplicate backend business rules or access the database, GLM, wallet,
  or chain directly.
- All displayed run data comes from backend APIs or SSE.
- Test loading, empty, failure, and approval states.
- Before finishing, run `scripts/check_dashboard.sh`.
